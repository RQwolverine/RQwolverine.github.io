Before we start

  • 四月五月一直处于极度忙碌中，这周末忙着复习电磁学…无暇更新网站，若不是今天发现网站因近一个月未更新导致相应速度明显变慢，也不会想到临时贴一次代码……今天所选的主题来自C You Coursework II，创作者为团体协作。

Description

  • Mandelbrot Set是一种分形图案，可以通过C语言代码实现，这个project的原创者: Dr. Aphrodite Galata, Manchester, UK.
  • 关于这个项目有一个英文版本的PDF介绍，你可以在Bing里检索到。

Analysis

  • 此project我们使用Codeblocks书写代码，在这之前我们配置OpenCV库，具体操作、环境变量配置、接入方式可以在Baidu上搜索到，我们利用OpenCV库中的函数，封装4个自定义函数，即可完成Mandelbrot Set分形图的绘制、用户鼠标键击缩放、重绘制三个功能。
